Timestamp Microservice

How to use

  • Provide a YYYY-MM-DD date or a UNIX date by accessing the /api endpoint!
  • Upon accessing the endpoint, you'll receive a JSON object containing a timestamp.

Example Usage:

  • https://timestamp-project.vercel.app/api/2015-12-25
  • https://timestamp-project.vercel.app/api/1451001600000

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
